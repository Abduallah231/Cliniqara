BR-CMP-001
Each visit can have only one chief complaint.

BR-CMP-002
Chief Complaint must exist in Complaint Library.

BR-CMP-003
Complaint Template is read-only.

BR-CMP-004
Analysis values are stored separately from template.

BR-CMP-005
Template version is saved with every visit.

BR-CMP-006
Changing a template must not affect previous visits.

BR-CMP-007
Duration value cannot exist without duration unit.

BR-CMP-008
If no template exists, Generic Analysis is used.

BR-CMP-009
Only answered fields are saved.

BR-CMP-010
Field codes are immutable.
